CeejOS: The Pseudo-VM
=====================

* * *

This website was created to pay homage to two things I love: Windows XP and Hayley Williams’ recent album, ‘Ego Death at a Bachelor Party.’ I wanted to pay homage to both of them by creating my own ‘operating system’ with the illusion of an active desktop through a static website.

Each section of the website can be accessed as if you’re navigating a desktop: by clicking on one of the open programs at the taskbar (or the Navbar) and opening a “window.” Be on the lookout for easter eggs within the site.
